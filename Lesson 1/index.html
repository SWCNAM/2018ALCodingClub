What is Computer Programming?
Computer programming is a way of giving computers instructions about what they should do next. These instructions are known as code, and computer programmers write code to solve problems or perform a task.
What Can You Make With Computer Code?
Computer code can be used to create anything that can run on a computer. This can be things like web sites, desktop applications (Microsoft Word, Google Chrome), phone applications, video games, and much more.
What is a Programming Language?
Programming languages are tools that we can use to create the code (computer instructions) that tell the computer what we want it to do.

Computers work by reading a series of 1s and 0s called binary data. Its really hard for humans to understand 1s and 0s, so engineers made programming languages.

A programming language lets us write code that is easy for a human to understand. The code is then compiled into 1s and 0s so that a computer can understand what to do.

There are many different programming languages to choose from
  • JavaScript
  • C++
  • C#
  • Python
  • Many more...
Most programming languages accomplish the same goal, but each language will be suited toward certain tasks. We have to decide what our task is, and which programming language best suits our needs to accomplish that task.


Which Programming Language Will We Be Using?
For the coding club, we will be using JavaScript

JavaScript is a language that is primarily used to make interactive web pages. When you go to any website, there is a good chance that it is using JavaScript to make the page function properly.

We will be writing JavaScript code inside of a .js file, and we can then view the output by going to a web browser and refreshing the page.
