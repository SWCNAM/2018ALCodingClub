Arrays
Arrays are a type of variable. They let us store multiple items in a single variable.
Variable Refresher
var name = "Jim";
Variables give us a way to store a value and give it a label so our program can easily refer to it later.
Think of a variable like a box that you can put something in. In our example, imagine that the box has "name" written on the side of it, and inside the box is the word "Jim".
Creating an Array
var names = ["Jim", "Katie", "Bill", "Adelaide"];
Use square brackets "[ ]" and separate each value with a comma ",".
var numbers = [200, 120, 3000, 450];
You can store any type of variable (strings, numbers, etc...) in arrays.
Getting Data Out of an Array
We cannot get values out of an array the same way we do out of a normal variable. We need to tell the computer which item in the array we want to get the value for. We do this by telling the computer the "index" that we want to use.
var names = ["Jim", "Katie", "Bill", "Adelaide"];
var name = names[0];
// name has the value of "Jim"

var name = names[1];
// name has the value of "Katie"

var name = names[5];
// There is no element at index 5, so this would
// not able to give us a value. This is a common issue with
// arrays so look out for this.
Why Do Arrays Start at 0 Instead of 1?
Think of an array like a line at the grocery store.

If you are the first person in line, how many people are in front of you? 0.

If you are the second person in line, how many people are in front of you? 1.

Modifying an Array
You can add to, remove from, or change items in an array
For loops
For loops let us run a piece of code multiple times for each item in a list
